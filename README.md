# Quoridor (现代网页版路障棋)

> **⚠️ 特别声明 (Notice)**
> 本项目仅供个人学习、研究与娱乐使用，**严禁用于任何商业目的或盈利行为**。

这是一个精美流畅的现代网页版 **路障棋 (Quoridor)** 游戏。Quoridor 是一款经典的策略棋盘游戏，目标是比对手先到达棋盘对面的底线。玩家在回合中可以选择移动棋子或放置木墙来阻挡对手。

![Quoridor Demo](screenshot.png)

## ✨ 项目特点 (Features)

* **🆚 本地双人对战**：支持在同一设备上进行双人面对面对局。
* **🤖 单机 AI 对战**：内置 AI 对手，支持“简单、普通、困难”三种难度调节。
* **📊 局势评估条**：实时计算并直观展示当前的胜率/局势走向。
* **📜 动作历史记录**：详细记录并展示每一步行棋和放置墙壁的动作历史。
* **⏪ 撤销与悔棋**：随时可以撤销上一步操作。
* **💡 AI 支招提示**：不知道怎么走时，可允许 AI 提供最佳移动提示。
* **⏱️ 倒计时功能**：支持类似国际象棋的 Fisher Clock 倒计时模式（每步加时）。
* **🎨 精美 UI 体验**：适配现代网页界面，带有流畅的移动与放置动画、音效支持及设备振动等“Juicy”的交互反馈。

## 🚀 运行与构建

1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
3. 构建项目：
   ```bash
   npm run build
   ```
